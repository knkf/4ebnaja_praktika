Для запуска проекта необходимо открыть файл main.py. 
Запуск должен производиться после того как проект открыт в среде PyCharm. 
Необходим Python 3.11 или свежее. 
Используемые библиотеки: 
PyQt5 5.15.9
pandas 2.0.2
openpyxl 3.1.2 
Команды для установки:
pip install pyqt5
pip install pandas
pip install openpyxl

Если команды для установки библиотек не работают, ошибка "Имя не распознано как имя командлета, функции, файла сценария или выполняемой программы", 
необходимо сделать следующее:
Изменить переменную среды PATH, или переустановить Python с установкой галочки в чекбокс "Add Python 3.11 to PATH". ВАЖНО: Python нужно именно переустанавливать,
а не обновлять для новой версии, так как иначе эта галочка не появится.

Если при попытке запуска файла main.py окно сразу же закрывается, необходимо произвести следующие процедуры:
1) Перейти по пути C:\Users\Имя_пользователя\AppData\Local\Programs\Python\Python311\Lib\site-packages\PyQt5\Qt5\plugins
Чтобы перейти в папку AppData пользуемся сочетанием клавиш Win+R
2) Найти в папке plugins папку platforms, скопировать её и переместить в папку с исполняемым файлом python.exe 
То есть по следующему пути C:\Users\Имя_пользователя\AppData\Local\Programs\Python\Python311
